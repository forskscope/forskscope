# Basic Diff Workflow

The diff workspace shows your two files side by side. Changed lines are
highlighted: deletions on the left, insertions on the right, replacements on
both sides.

The position indicator (e.g. **2 / 5**) shows which change you are on out of
how many. Use ◀ ▶ to step through them in order.

## Inline character diff

When you want to see exactly which characters changed within a line, click
**More ▼** in the toolbar to reveal the **Inline diff** toggle.  Turn it on;
the changed characters within each replaced line become highlighted.  Turn it
off to return to the clean line view.

## Reading the gutter marks

A ▶ button in the action column means this hunk can be applied (left → right).
A ✓ mark means the hunk has already been applied in this session.
