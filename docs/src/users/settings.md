# Settings

Click **Settings** in the header to open the settings panel.

| Setting | Options | Effect |
|---|---|---|
| Theme | Dark (default) / Light / Night | Changes the colour scheme. |
| Language | English / Japanese | Changes the UI language. |
| Diff font size | 8–32 px | Controls the size of text in the diff panes. |

Settings are saved immediately when changed and restored the next time you
launch the app.
