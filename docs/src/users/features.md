# Features

## Core workflow

- **Side-by-side text comparison** with line-level diff.
- **Hunk navigation** — move between changes with ◀ ▶; never scroll manually through large files.
- **Selective merge** — apply individual changes from the left (old/source) side to the right (new/target) side; skip changes you do not want.
- **Undo** — every merge action is reversible.
- **Safe save** — external-modification detection before writing; an automatic `.bak` backup; atomic write.

## Explorer

- Two directory panes with lazy-loading file trees.
- Select a file on each side and click **Compare** to open a diff tab.
- Multiple comparisons open as tabs; switch between them freely.

## File types

| Type | Diff | Merge / Save |
|---|---|---|
| Text (UTF-8, Shift_JIS, …) | ✓ | ✓ |
| Excel `.xlsx` | ✓ (read-only) | — |
| Binary / other | Hex preview | — |

## Appearance

- Three built-in themes: dark (default), light, night.
- Configurable diff font size.
- Japanese and English localization.

## Privacy

No accounts, no telemetry, no cloud upload.  Everything stays on your machine.
