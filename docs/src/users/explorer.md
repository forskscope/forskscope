# Explorer Panes

The Explorer workspace is the starting point.  It has two directory panes side
by side: left (old/source) and right (new/target).

## Navigate directories

Type a path in the path bar and click **List**, or click a folder in the tree
to expand it.

## Open a comparison

Click a file in the left pane to select it as the left candidate, then click a
file in the right pane to select it as the right candidate.  Click **Compare**
to open a diff tab.

## Multiple tabs

Each comparison opens in its own tab.  Click any tab to switch to it.
Click ✕ on a tab to close it.
